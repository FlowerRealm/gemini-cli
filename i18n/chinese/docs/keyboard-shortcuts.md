# 键盘快捷键

Gemini CLI 支持多种键盘快捷键，可帮助您更高效地工作。

## 全局快捷键

| 快捷键 | 操作 |
| --- | --- |
| `Ctrl+C` | 中止当前命令 |
| `Ctrl+D` | 退出 CLI |
| `Ctrl+L` | 清除屏幕 |

## 编辑器快捷键

在多行编辑器中输入提示时，可以使用以下快捷键：

| 快捷键 | 操作 |
| --- | --- |
| `Alt+Enter` | 提交提示 |
| `Up Arrow` | 移至上一行 |
| `Down Arrow` | 移至下一行 |
